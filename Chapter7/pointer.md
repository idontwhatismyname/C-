# 指针
## 指针和数组的关系
### 一级指针和一维数组的关系
int num[10]等价于int *const num  
num作为一个常量指针存在
### 二级指针和二维数组的关系
#### int ** ptr与int ptr[m][n]
由于*ptr指向的是一个int类型的指针，其指向的是一个指针 *ptr 
而ptr[]指向的是一个长度为n的数组  
故二维数组应该对应int (*ptr)[n]
## 理解一个指针的方式
举例子int (*a)[10]  
第一个括号决定了a是一个指针，后面的[10]决定了它是指向长度为10的数组  
另一个例子 int *a[10]  
按照优先级顺序，a首先是一个长度为10的数组，其元素均为(int *)类型
简单来说,找到先确定变量类型,再将(*变量名)视为一个变量名的代替品即可
